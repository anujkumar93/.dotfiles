
# Reading

| command    | description |
|------------+-------------|
| `ctrl + e` | scroll down |
| `ctrl + y` | scroll up   |

# Convenient Functions

| command | description   |
|---------+---------------|
| `gg`    | go to the top |
| `gx`    | open a link   |
| `gq`    | wrap line     |


# Autocompletion

| command      | description                  |
|--------------+------------------------------|
| `<C-X><C-K>` | autocomplete from dictionary |
